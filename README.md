# 3D Model Viewer

Simple web project for displaying 3D models made in Blender and rendered in the browser.

## Live Demo

- Main page: https://dimitrijestamenkovic-fmt.netlify.app/
- Model 2: https://dimitrijestamenkovic-fmt.netlify.app/model2
- Model 3: https://dimitrijestamenkovic-fmt.netlify.app/model3

## About

This project displays custom 3D models created in Blender and loaded in the browser using Three.js.

## Features

- Multiple 3D models
- Navigation between different scenes/pages
- Interactive model viewing in browser

## Tech Stack

- Three.js
- Blender
- JavaScript
- HTML/CSS
- Netlify

## Models

All 3D models were created in Blender.
